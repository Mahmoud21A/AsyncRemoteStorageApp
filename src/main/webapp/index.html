- Project Description: -

We would like to take control of a shared folder under a remote system.

More specifically, we would like to be able to:

  • Browse the remote shared folder, including its sub-folders.
  • Rename a remote shared file or sub-folder.
  • Download a remote shared file.
  • Upload a local file to a specified path under the remote shared folder.
  • Delete a shared remote file. (Deleting a sub-folder is not allowed.)

- How to Use: -

1.Select the file/folder you wish to perform an operation on by clicking on it

then press on the operation button (Browse/Download, Rename, Delete).

2.To upload a file, press on the upload button, choose the file you wish to upload,

then specify a path. For example: "folderA/folderD" (Without the quotes).

- Known Bugs: -

1.Renaming a file inside a sub-folder fails.

2.Attempting to navigate to a sub-folder immediately after renaming it fails, but

reloading the page does fix the issue.
